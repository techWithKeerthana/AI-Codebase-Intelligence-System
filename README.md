# AI Codebase Intelligence System

## Overview

AI Codebase Intelligence System is an AIML project that analyzes complete software repositories using source-code parsing, GraphRAG, vector databases, and Large Language Models.

## Phase 1

- Repository analysis
- Source code parsing
- AST extraction
- Dependency graph generation
- Architecture visualization

## Technology Stack

- Python
- FastAPI
- Tree-sitter
- GitPython
- NetworkX
- GraphRAG
- ChromaDB
- Gemini/OpenAI API
